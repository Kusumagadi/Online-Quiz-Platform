# Online-Quiz
Regarding Online Quiz Platform in React

Git Workflow Steps
1. Clone the repository

Make sure Git Bash is installed on your system.
Then run the following command to clone the project:

git clone https://github.com/KomalSonawane15699/Online-Quiz.git

2. Create a new branch (Do NOT commit directly to main)

Create your working branch from the main branch:

git checkout -b online-quiz-komal origin/main

3. Commit your changes

After making changes, use the following commands:

git add <file-path>
git commit -m "your commit message"
git push origin online-quiz-komal

4. Pull the latest changes from main

Always pull new updates from the main branch before starting new work:

git pull origin main
